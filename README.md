# 基于51单片机的智能循迹小车

## 实验概述

本实验基于 AT89C51 单片机 + L293D 电机驱动芯片，实现双轮差速智能小车的**红外循迹**功能。通过两路红外对管检测地面黑线，配合软件 PWM 调速，使小车自动沿黑色轨迹行驶。

## 实验目的

1. 掌握单片机通过 H 桥（L293D）控制直流电机正反转的方法
2. 掌握软件 PWM 调速原理及定时器中断实现方法
3. 理解双轮差速转向的循迹控制算法
4. 掌握红外传感器信号采集与处理
5. 掌握 Proteus 仿真环境中智能小车系统的搭建与调试

## 硬件架构

```
                    +----------+
                    | AT89C51  |
                    |          |
   启动按键 ------->| P1^6     |
                    |          |
   左红外传感器 -->| P3^4(DL) |      +---------+
   右红外传感器 -->| P3^5(DR) |      | L293D   |
                    |          |      |         |
                    | P1^5 ----|----->| EN1,2   |----> 左电机
                    | P1^4 ----|----->| 1A      |
                    | P1^3 ----|----->| 2A      |
                    |          |      |         |
                    | P1^0 ----|----->| EN3,4   |----> 右电机
                    | P1^2 ----|----->| 3A      |
                    | P1^1 ----|----->| 4A      |
                    +----------+      +---------+
```

## 循迹控制算法

采用**双路红外比例修正**策略：

| 传感器状态 (DL, DR) | 小车状态 | 修正策略 |
|---------------------|---------|---------|
| (0, 0) | 在黑线上 | 直行，PWM=180 |
| (1, 0) | 偏左 | 右转修正，右PWM=50 / 左PWM=220 |
| (0, 1) | 偏右 | 左转修正，右PWM=220 / 左PWM=50 |
| (1, 1) | 完全偏离 | 减速搜索，PWM=120 |

> 注：红外对管检测到黑线输出低电平(0)，白色地面输出高电平(1)。

## 软件 PWM 实现

- **定时器**：T0 工作模式2（8位自动重装），50μs 中断周期
- **PWM 周期**：256 × 50μs = 12.8ms（约 78Hz）
- **占空比精度**：1/256 ≈ 0.4%
- **调速范围**：PWM 值 0~255，对应占空比 0%~100%

- **晶振**：12MHz

## 实验步骤

1. 在 Proteus 中新建工程，添加 AT89C51 单片机最小系统（晶振 + 复位电路 + 上拉电阻）
2. 添加 L293D 电机驱动芯片及两个直流电机
3. 添加两路红外对管传感器，输出接 P3^4 / P3^5
4. 按 `SCHEMATIC.md` 中的连接表完成全部接线
5. 在 Keil 中编译 `main.c`，生成 HEX 文件
6. 将 HEX 加载到 Proteus 中的 AT89C51
7. 按下启动按键，观察小车循迹行为
8. 调整 PWM 修正参数（PwmR/PwmL），优化过弯性能
